AngularJS - Pros & Cons
Pros Cons
  • Uses JavaScript; hence easy to learn
  • Two-way data binding
  • Fast development for small applications
  • No dynamic loading
  • No CLI

  • Angular vs AngularJS (Virginia Tech - CS3304: Comparative Languages, Fall 2020)
Angular 2 - Pros & Cons
Pros Cons
  • Client tool for expedited development.
  • Component Decorators.
  • Event Bindings.
  • Code Base overhauled as compared to that of AngularJS -> i.e. AngularJS Users had to relearn syntax.

  • Angular vs AngularJS (Virginia Tech - CS3304: Comparative Languages, Fall 2020)
